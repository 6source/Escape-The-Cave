# Escape The Cave

![License](https://img.shields.io/badge/license-MIT-green)

## 📝 Description

Escape-The-Cave is an adventure game where you try to escape a dark underground cave. 
You solve puzzles, avoid traps, and explore the cave to find the way out. 
The challenges get harder as you go, and every move matters if you want to reach the exit.
